Сетевое взаимодействие с использованием сокетов в Python (TCP)

В этой документации представлено два простых приложения на Python, демонстрирующие коммуникацию по протоколу TCP (Transmission Control Protocol): клиент и сервер. Клиент отправляет коэффициенты для квадратного уравнения, а сервер решает уравнение и возвращает результат.

Обзор

TCP — это протокол передачи данных, который обеспечивает надежную передачу сообщений между клиентом и сервером. Эта документация охватывает два основных компонента:

  1. TCP Клиент: Клиентское приложение, которое запрашивает у пользователя коэффициенты квадратного уравнения и отправляет их на сервер.
  2. TCP Сервер: Серверное приложение, которое принимает коэффициенты, решает квадратное уравнение и возвращает результат клиенту.

Требования

Для запуска приложений убедитесь, что у вас установлен Python. Код использует встроенную библиотеку socket, которая доступна в стандартных дистрибутивах Python, а также библиотеку math для математических операций.

Код Клиента

Следующий раздел описывает код TCP клиента.

python
import socket

# Создаем сокет
client_socket = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
client_socket.connect(('localhost', 12345))

try:
    # Запрашиваем параметры у пользователя
    a = float(input("Введите коэффициент a: "))
    b = float(input("Введите коэффициент b: "))
    c = float(input("Введите коэффициент c: "))

    # Отправляем данные серверу
    client_socket.send(f"{a} {b} {c}".encode())

    # Получаем результат от сервера
    result = client_socket.recv(1024).decode()
    print(f"Результат: {result}")
except KeyboardInterrupt:
    print("\nКлиент остановлен.")
except Exception as e:
    print(f"Ошибка: {str(e)}")
finally:
    client_socket.close()

Код Сервера

import socket
import math

def solve_quadratic(a, b, c):
    D = b ** 2 - 4 * a * c
    if D > 0:
        x1 = (-b + math.sqrt(D)) / (2 * a)
        x2 = (-b - math.sqrt(D)) / (2 * a)
        return f"Два решения: x1 = {x1}, x2 = {x2}"
    elif D == 0:
        x = -b / (2 * a)
        return f"Одно решение: x = {x}"
    else:
        return "Нет действительных решений"

# Создаем сокет
server_socket = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
server_socket.bind(('localhost', 12345))
server_socket.listen(1)

print("Сервер запущен и ожидает подключения...")

try:
    while True:
        conn, addr = server_socket.accept()
        print(f"Подключено к {addr}")

        try:
            data = conn.recv(1024).decode()
            a, b, c = map(float, data.split())
            result = solve_quadratic(a, b, c)
            conn.send(result.encode())
        except Exception as e:
            conn.send(f"Ошибка: {str(e)}".encode())
        finally:
            conn.close()
except KeyboardInterrupt:
    print("\nСервер остановлен.")
finally:
    server_socket.close()
