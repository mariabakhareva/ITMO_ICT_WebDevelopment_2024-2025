Forms

Этот файл содержит формы для работы с приложением "Доска Заданий". Формы управляют вводом данных от пользователей для регистрации, входа, отправки заданий и создания новых заданий.

StudentRegistrationForm

StudentRegistrationForm — форма регистрации нового студента.

Поля

  • username: CharField
  • Имя пользователя для регистрации студента.
  • name: CharField
  • Полное имя студента.
  • email: EmailField
  • Уникальный email для студента.
  • pwd: CharField
  • Поле для ввода пароля. Использует PasswordInput для скрытия ввода.
  • confirm_pwd: CharField
  • Поле для подтверждения пароля. Использует PasswordInput для скрытия ввода.

Методы

  • clean: проверяет, совпадают ли пароли pwd и confirm_pwd. Выдает ошибку ValidationError, если пароли не совпадают.
  • save: сохраняет студента с зашифрованным паролем, используя метод set_password модели Student.

StudentLoginForm

StudentLoginForm — форма для входа зарегистрированного студента.

Поля

  • username: CharField
  • Имя пользователя студента.
  • password: CharField
  • Пароль студента. Использует PasswordInput для скрытия ввода.

Описание

Форма StudentLoginForm используется для аутентификации студентов при входе в систему. Она принимает имя пользователя и пароль для проверки учетных данных.

AssignmentSubmissionForm

AssignmentSubmissionForm — форма для отправки выполненного задания.

Поля

  • submission_text: TextField
  • Текстовое поле, в которое студент вводит свой ответ на задание.

Описание

Форма позволяет студентам отправлять текстовый ответ на задание. Форма связана с моделью Assignment и используется при отправке выполненного задания на проверку.

TaskCreationForm

TaskCreationForm — форма для создания нового задания.

Поля

  • title: CharField
  • Название задания (максимальная длина — 200 символов).
  • description: TextField
  • Полное текстовое описание задания.
  • due_date: DateField
  • Дата завершения задания. Использует SelectDateWidget для удобного выбора даты.

Методы

  • Meta: определяет связь с моделью Task и поля title, description, и due_date.

Описание

Форма TaskCreationForm позволяет преподавателю или администратору создавать новое задание с указанием названия, описания и крайнего срока сдачи.
