Регистрационная форма

Этот компонент React представляет собой форму регистрации, которая позволяет пользователю создать новый аккаунт. Включает обработку данных формы и отправку их на сервер с использованием библиотеки axios. После успешной регистрации пользователь перенаправляется на страницу входа.

Основная функциональность

Состояние формы

Компонент использует хук useState для хранения данных формы, включая следующие поля: - username: имя пользователя - email: адрес электронной почты - password: пароль - re_password: подтверждение пароля

Также используется состояние для отображения сообщений об ошибке (error) и успешной регистрации (success).

Обработка изменений в форме

Каждое изменение в полях формы обрабатывается функцией handleChange, которая обновляет соответствующее поле в состоянии formData.

Отправка данных на сервер

При отправке формы вызывается функция handleSubmit, которая выполняет POST-запрос на сервер по адресу http://127.0.0.1:8000/api/auth/users/ с данными формы. Если регистрация успешна, пользователю показывается сообщение о успехе, а затем происходит редирект на страницу входа. В случае ошибки отображается сообщение с ошибкой, полученной от сервера.

Перенаправление после успешной регистрации

После успешной регистрации пользователь перенаправляется на страницу входа через navigate("/login"). Переход осуществляется через 2 секунды после получения успешного ответа от сервера.
