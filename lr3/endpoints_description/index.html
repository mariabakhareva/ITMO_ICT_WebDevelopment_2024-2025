Документация по эндпоинтам API

Работа с пациентами

POST /api/v1/patients/

Аргументы: - Тело запроса: json { "first_name": "string", "last_name": "string", "birth_date": "YYYY-MM-DD", "phone_number": "string", "email": "string", "address": "string" }

Описание: Создаёт нового пациента. Данные, переданные в теле запроса, сохраняются в базе данных.

Ответы: - 201: Успешное создание пациента. Возвращает данные созданного пациента. - 400: Ошибка валидации. Возвращает список ошибок.

GET /api/v1/patients/

Аргументы: Нет.

Описание: Возвращает список всех пациентов.

Ответы: - 200: Успешное выполнение. Возвращает массив объектов пациентов.

GET /api/v1/patients/{id}/

Аргументы: - Параметры пути: - id (integer): Идентификатор пациента.

Описание: Возвращает информацию о конкретном пациенте.

Ответы: - 200: Успешное выполнение. Возвращает объект пациента. - 404: Пациент с указанным id не найден.

PUT /api/v1/patients/{id}/

Аргументы: - Параметры пути: - id (integer): Идентификатор пациента. - Тело запроса: json { "first_name": "string", "last_name": "string", "birth_date": "YYYY-MM-DD", "phone_number": "string", "email": "string", "address": "string" }

Описание: Обновляет информацию о пациенте с указанным id.

Ответы: - 200: Успешное обновление. Возвращает обновлённый объект пациента. - 400: Ошибка валидации. Возвращает список ошибок. - 404: Пациент с указанным id не найден.

DELETE /api/v1/patients/{id}/

Аргументы: - Параметры пути: - id (integer): Идентификатор пациента.

Описание: Удаляет пациента с указанным id.

Ответы: - 204: Успешное удаление. - 404: Пациент с указанным id не найден.

Кастомные отчёты

GET /api/v1/reports/patients-by-doctor/

Аргументы: - Параметры запроса: - doctor_id (integer): Идентификатор врача.

Описание: Возвращает список пациентов указанного врача с датами и стоимостью их визитов.

Ответы: - 200: Успешное выполнение. - 400: Некорректные параметры запроса.

GET /api/v1/reports/phones-by-condition/

Аргументы: Нет.

Описание: Возвращает список телефонов пациентов, которые посещали отоларинголога и родились после 1987 года.

Ответы: - 200: Успешное выполнение.

GET /api/v1/reports/doctors-by-date/

Аргументы: - Параметры запроса: - date (YYYY-MM-DD): Дата.

Описание: Возвращает список врачей, работающих в указанный день.

Ответы: - 200: Успешное выполнение. - 400: Некорректные параметры запроса.

GET /api/v1/reports/visit-counts-by-date/

Аргументы: Нет.

Описание: Возвращает количество визитов по датам.

Ответы: - 200: Успешное выполнение.

GET /api/v1/reports/costs-by-date-and-doctor/

Аргументы: Нет.

Описание: Возвращает суммарную стоимость лечения пациентов, сгруппированную по дням и врачам.

Ответы: - 200: Успешное выполнение.

GET /api/v1/reports/paid-patients/

Аргументы: Нет.

Описание: Возвращает список пациентов, которые уже оплатили лечение.

Ответы: - 200: Успешное выполнение.

GET /api/v1/reports/doctor-work-details/

Аргументы: - Параметры запроса: - start_date (YYYY-MM-DD): Начальная дата периода. - end_date (YYYY-MM-DD): Конечная дата периода.

Описание: Возвращает подробный отчёт о работе врачей за указанный период, включая общий доход и информацию о визитах.

Ответы: - 200: Успешное выполнение. - 400: Некорректные параметры запроса.
